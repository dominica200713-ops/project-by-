# Бюджетный помощник

Проект оформлен по тематическим блокам. В `main.py` перед основными
структурами данных, алгоритмами и элементами интерфейса размещены простые
поясняющие комментарии с символом `#`.

## 1. Название проекта

«Бюджетный помощник», вариант № 7 по дисциплине «Основы программирования».

## 2. Цель проекта

Программа помогает пользователю учитывать расходы за месяц, добавлять траты
по дням и категориям, быстро считать сумму расходов за выбранный период и
анализировать расходы.

## 3. Возможности программы

- добавление расхода с указанием дня, суммы и категории;
- проверка правильности введённых данных;
- просмотр списка всех добавленных расходов;
- расчёт суммы расходов за период;
- поиск дня с максимальной суммой расходов;
- вывод категорий по убыванию общей суммы трат;
- отмена последнего добавленного расхода;
- вывод расходов из дерева по возрастанию дней.

## 4. Используемые алгоритмы и структуры данных

- массив расходов по дням хранит общую сумму для каждого из 31 дня;
- массив префиксных сумм позволяет отвечать на запрос за период за `O(1)`;
- линейный поиск находит день с максимальными расходами без `max()`;
- сортировка вставками упорядочивает категории без `sorted()` и `.sort()`;
- стек хранит расходы в порядке добавления для отмены последней операции;
- бинарное дерево поиска хранит расходы с ключом, равным дню месяца;
- рекурсивный симметричный обход дерева выводит дни по возрастанию.

## 5. Распределение работы между Софьей и Доминикой

София отвечает за классы и структуры данных, массив расходов по дням, список
расходов, стек отмены, дерево, префиксные суммы, расчёт суммы за период,
линейный поиск и сортировку вставками.

Анастасия отвечает за окно `tkinter`, голубое оформление, поля ввода, кнопки,
список расходов, обработчики действий, вывод результатов и запуск приложения.

В `main.py` части разделены комментариями `# часть софьи` и
`# часть доминики`. Внутри частей код дополнительно разделён на блоки:

- запись об одном расходе;
- узел бинарного дерева;
- бинарное дерево расходов;
- основная логика бюджетного помощника;
- главное окно программы;
- запуск приложения.

Каждое важное действие подписано простым комментарием: создание массивов,
работа стека, построение префиксных сумм, линейный поиск, сортировка вставками,
рекурсивный обход дерева и обработка кнопок.

## 6. Как запустить проект

Требуется Python 3 с поддержкой стандартной библиотеки `tkinter`.

```text
cd D:\Project\budget_helper
python main.py
```
